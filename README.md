# Build your first .NET console app

_Learn the fundamentals of creating, running, and debugging a .NET console application in a prepared workspace._

## Welcome

- **Who is this for**: Beginners who want hands-on practice creating their first .NET console application in GitHub.
- **What you'll learn**: How to set up a C# workspace, scaffold a console app, add input/output behavior, and validate with run/debug workflows.
- **What you'll build**: A beginner-friendly .NET console app that accepts user input and prints a personalized response.
- **Prerequisites**:
  - Basic familiarity with files, folders, and terminal commands.
  - Ability to open a repository in GitHub Codespaces or VS Code.

- **How long**: This exercise takes less than 45 minutes to complete.

In this exercise, you will:

1. Prepare the workspace with C# Dev Kit and .NET SDK prerequisites.
1. Create a new console app from a template.
1. Modify `Program.cs` to accept input and produce output.
1. Run and debug the app in VS Code.
1. Open a pull request with your completed console app.
